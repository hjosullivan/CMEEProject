## Installation
1. Clone repo, e.g.: `git clone ...`
2. Create virtualenv, e.g.: `virtualenv -p python3 venv`
3. Activate virtualenv, e.g.: `source venv/bin/activate`
4. Install requirements, e.g.: `pip install -r requirements.txt`
